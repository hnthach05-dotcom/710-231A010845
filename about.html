Giới thiệu

Đây là website bài tập thực hành môn Lập trình Web. Website được xây dựng bằng HTML, CSS và JavaScript, triển khai trên nền tảng GitHub Pages.

Mục tiêu của website là rèn luyện kỹ năng xây dựng giao diện, xử lý sự kiện JavaScript và deploy website lên hosting.
